# Vanguard Attention Intelligence System (SIM_LIVE_v3.0)

An elite full-stack system designed to execute **Attention Discovery, Trigger Auditing, and Hook Optimization** for modern digital campaigns using the **Vanguard Strategic SOP**.

Vanguard shifts the focus upstream: rather than simply formatting final text, the platform discovers high-potential **psychological hooks and tactical angles** *before* fine-tuning copy. It enforces the concept **"Attention first, Optimization second."**

---

## 🚀 Technical Workflow

### 1. 🔍 The Attention Broker Engine (Upstream Discovery)
The campaign starts with identifying attention potential rather than drafting content. Input any raw topic or article to discover:
*   **Cognitive Trigger Auditing:** Calculates gravitational pulls for ten primary psychological triggers (e.g., *Curiosity, Conflict, Consequence, Hidden Incentives, Betrayal, Mystery, Status, Warning*).
*   **The Strategic Angle Library:** Dynamically discovers and builds exactly five strategic directions (e.g., *Who Benefits?*, *Historical Parallel*, *The Wrong Question*, *Myth vs Reality*, *Contrarian Position*).
*   **Opportunity Score Engine:** Scores each angle based on `Attention Score (Curiosity & Debate Potential) × Optimization Score (Viral Action Readiness)` and ranks them to select high-probability winners first.

### 2. ⚡ The SOP Optimizer Engine (Downstream Optimization)
Once a strategic angle is locked in, the system initiates the **90th Percentile SOP Audit**:
*   **Base Success Score (BSS) [Elite Standard ≥ 0.90]:** Evaluates conversation density, retention/dwell mechanics, and velocity potential.
*   **Keyword Strength Multiplier (KSM) [Elite Standard ≥ 0.95]:** Verifies structural pacing, frontloaded pattern-interrupt placements, and platform-native formatting.
*   **Unified Success Probability [Elite Target ≥ 0.85]:** Formulates real-world viability metrics to safeguard against low-value or generic output.

### 3. 🎯 The 3-Signal Loader Layer
Validates content drafts to ensure they load the vital triggers necessary for immediate viral acceleration:
*   **🔄 Rewatch Trigger:** Seamless loops, "Save this Checklist" cues, or bookmarkable resource lists.
*   **💬 Conversation Trigger:** High-friction polarizing debate-prompters to drive immediate reply volume.
*   **🔗 Share Trigger:** Clear forwarding incentives ("Send this to a friend who needs this") to break out of single-network silos.

### 4. 🧪 Vanguard Interactive Sandbox & Visualizers
*   **Real-time KPI Dials:** Live text area checks word counts, digit-triggers, and instant BSS/KSM feedback.
*   **AI Refiner Panel:** Instantly auto-tunes baseline copy into distinct platform styles like *Curiosity Loops, Loss Aversion, Authority Hubs*, and *Counter-Intuitive Sparks*.
*   **Multidimensional Timelines:** Visualizes the *Phenix Time-Decay Matrix*, *60-Second Audience Retention Map*, and anticipated *12h Production Projection Curves*.
*   **Live Screen Viewports:** Fully responsive mobile and desktop mockup frames representing native X/Twitter, TikTok, Instagram Reels, and YouTube Shorts interfaces.

## 🛠️ Technical Architecture & Stack

*   **Frontend:** [React 18](https://react.dev/) + [Vite](https://vite.dev/) dev server + [Tailwind CSS](https://tailwindcss.com/) for a sleek dark console experience.
*   **Backend:** [Express](https://expressjs.com/) server routing requests and managing secure, lazy-loaded client connections.
*   **AI Integration:** Modern `@google/genai` TypeScript SDK deploying parent model intelligence.
*   **Micro-Animations:** Fluid view switching and progress bar transitions managed by `motion` (`motion/react`).
*   **Typography:** [Inter](https://fonts.google.com/specimen/Inter) for administrative structures and data layout, paired with [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) for analytical calculations and score tracking.

## 📦 Local Installation & Setup

### 1. Set Up Environment Variables
Create a standard `.env` configuration file in the project root:
```env
GEMINI_API_KEY=your-gemini-api-key-here
```

### 2. Install Project Dependencies
Pull down and install node modules:
```bash
npm install
```

### 3. Run in Development Mode
Launch the live development server on external port `3000`:
```bash
npm run dev
```

### 4. Build for Production Optimization
Bundle both client-side static assets and compile backend TypeScript resources into a production-ready CJS build file:
```bash
npm run build
```

Then start the production build:
```bash
npm start
```
